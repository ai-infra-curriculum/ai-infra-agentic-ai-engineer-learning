# mod-202-frameworks: Agent Frameworks in Practice

> Scaffolded by `aicg org execute-plan`. Lecture chapters and exercise content are authored on subsequent autonomous cycles.

**Estimated effort:** 14 hours

## Learning objectives

- Build the same agent across LangGraph, CrewAI, and AutoGen and compare tradeoffs
- Use stateful graph workflows, role-based crews, and conversational patterns
- Evaluate and select between competing SDKs (OpenAI Agents SDK, Google ADK, Anthropic Agent SDK, smolagents)
- Apply MCP for standardized tool use

## Structure

- `01-…md` … `0N-…md`: lecture chapters.
- `exercises/`: per-exercise prompts.
- `labs/`: long-form hands-on labs.
- `quizzes/`: knowledge checks.
- `resources.md`: external references.
