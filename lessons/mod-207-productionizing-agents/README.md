# mod-207-productionizing-agents: Productionizing Agents

> Scaffolded by `aicg org execute-plan`. Lecture chapters and exercise content are authored on subsequent autonomous cycles.

**Estimated effort:** 12 hours

## Learning objectives

- Deploy agents behind APIs (FastAPI) with containerization
- Implement durable execution and resumption for long-running tasks
- Apply prompt caching and model routing for cost/latency
- Add human-in-the-loop with state persistence (e.g., LangGraph interrupt)
